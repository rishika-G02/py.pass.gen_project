PROJECT TITLE:
    Py_Pass_Gen

		

OVERVIEW:

A simple Python program that generates strong random passwords. Usera can choose
password length and include/exclude digits, symbols, uppercase, or lowercase letters. 
Designed to run interactively in Google Colab.



FEATURES:

*Generate strong random passwords

*Set custom password length

*Include/exclude digit, symbols,uppercase, and lowercase

*Work interactively in Colab



TOOL USED:

*Python 3.x

*Google Colab



STEPS TO INSTALL AND RUN THE PROGRAM:

Open Google Colab, create a new notebook, and write the code,run it,then
enter your choices and get a random password instantly.
